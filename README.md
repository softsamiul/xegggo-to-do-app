# Xeggo To Do App

##### Live url: https://xeggo-to-do.web.app/

#### Used Tech:
    1. React JS
    2. React Router Dom
    3. Redux
    4. Sass
    5. React hook Form
    6. Firebase

#### Overview: 
    This is a simple to do app. Here user can register by using email and password and also gmail account. The app is responsive for mobile and desktop devices.
